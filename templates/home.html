Accueil : Segmentation des Clients Bancaires

Bienvenue dans ce projet de Data Analytics et de Machine Learning pour la prédiction de l’attrition des clients bancaires.

Description du Dataset

Ce dataset contient {{ nb_clients }} enregistrements (clients). Les principales colonnes sont : {{ colonnes }}.

Contenu du dataset : - Détails démographiques (âge, genre, statut marital, catégorie de revenu).
- Informations sur la relation carte de crédit (type de carte, nombre de mois depuis l’ouverture, périodes d’inactivité).
- Comportement de dépense (solde total, limite de crédit, taux d’utilisation moyen, etc.).
- Indicateurs de churn (attrition) et variables possibles pour la prédiction.

Ce projet vise à analyser ces données et à construire un modèle pour prédire quels clients risquent de churner (fermer leur compte).
